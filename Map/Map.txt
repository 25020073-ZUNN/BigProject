📦 Hệ thống Đấu giá Trực tuyến (Online Auction System)
┣━ 📂 Nhóm Mô hình Dữ liệu (Models / Entities)
┃  ┣━ 📜 Entity (Lớp trừu tượng cơ sở - chứa ID, createdAt)
┃  ┃  ┃
┃  ┃  ┣━ 👤 User (Lớp trừu tượng)
┃  ┃  ┃  ┣━ 🧑‍💻 Bidder (Người mua - chứa balance, bidHistory)
┃  ┃  ┃  ┣━ 🏪 Seller (Người bán - chứa listedItems)
┃  ┃  ┃  ┗━ 🛠️ Admin (Quản trị viên)
┃  ┃  ┃
┃  ┃  ┣━ 📦 Item (Lớp trừu tượng)
┃  ┃  ┃  ┣━ 💻 Electronics (Đồ điện tử)
┃  ┃  ┃  ┣━ 🎨 Art (Nghệ thuật)
┃  ┃  ┃  ┗━ 🚗 Vehicle (Xe cộ)
┃  ┃  ┃
┃  ┃  ┣━ ⚖️ Auction (Phiên đấu giá - Quản lý Item, trạng thái, thời gian)
┃  ┃  ┗━ 💸 BidTransaction (Giao dịch đặt giá)
┃  ┃
┃  ┗━ ⚙️ AutoBidConfig (Cấu hình cho tính năng Auto-Bidding)
┃
┣━ 📂 Nhóm Thiết kế Hệ thống & Mẫu (Design Patterns)
┃  ┣━ 🏭 ItemFactory (Factory Method - Chịu trách nhiệm khởi tạo Item)
┃  ┃
┃  ┣━ 👑 AuctionManager (Singleton - Quản lý tập trung các phiên đấu giá đang chạy)
┃  ┃
┃  ┣━ 👁️ Nhóm Realtime Update (Observer Pattern)
┃  ┃  ┣━ 📜 BidSubject (Interface - Đối tượng bị quan sát, ví dụ: Auction)
┃  ┃  ┗━ 📜 BidObserver (Interface - Kẻ quan sát, ví dụ: Client GUI / JavaFX Controller)
┃  ┃
┃  ┗━ 🧠 Nhóm Xử lý Đặt giá (Strategy Pattern)
┃     ┣━ 📜 BidStrategy (Interface)
┃     ┣━ 🖐️ ManualBidStrategy (Logic đặt giá bằng tay)
┃     ┗━ 🤖 AutoBidStrategy (Logic tự động nhảy giá)
┃
┗━ 📂 Kiến trúc MVC (Phân tách Client & Server)
     ┣━ 🖥️ View (Giao diện JavaFX - .fxml)
     ┃  ┣━ Màn hình Đăng nhập / Đăng ký
     ┃  ┣━ Màn hình Quản lý Sản phẩm (Cho Seller)
     ┃  ┗━ Màn hình Đấu giá Trực tiếp (Realtime Bidding - Cập nhật biểu đồ giá)
     ┃
     ┣━ 🎮 Controller (Xử lý sự kiện từ UI)
     ┃  ┗━ Bắt sự kiện click, gọi Data qua Server/API
     ┃
     ┗━ 🗄️ DAO / Repository (Data Access Object)
            ┗━ Tầng tương tác trực tiếp với Database để lưu trữ dữ liệu bền vững
